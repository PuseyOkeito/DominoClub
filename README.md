# Domino Club Landing Page

A Next.js application for managing domino game sessions, matchmaking, and player assignments.

## Features

- Player registration and check-in
- Session management with configurable start times
- Automatic matchmaking (partnered and solo players)
- Table assignment (max 6 tables per session, 24 players)
- Real-time waiting room with timer
- Admin dashboard for game management
- Reset session functionality

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Supabase (PostgreSQL + Realtime)
- Tailwind CSS
- shadcn/ui components

## Deployment

Deployed on Vercel with automatic deployments from GitHub.
